trans-merge-punct

(trans-merge-punct tokens)

In seq tokens merges those groups that contain only punctuation. E.g., [ “Wow” “!” “!” “!” ] -> [ “Wow” “!!!” ]. Inverse of trans-split-punct.

trans-split-punct

(trans-split-punct tokens)

Split all punctuation tokens from tokens into separate characters. E.g., [ “Wow” “!!!” ] -> [ “Wow” “!” “!” “!” ] Inverse of trans-split-punct.
